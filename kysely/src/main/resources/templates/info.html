API End Pointit

Kaikkien tietojen hakemista varten

/kyselyt/1/kysymykset

RequestMethod.GET eli saa tiedot palvelimelta.

Näkee kyselyn nimen ja kaikki kyselyn kysymykset ja vastaukset mikäli niitä on tallennettu.

Numeron kohdalle laitetaan kyselyn id (Kahvi-kyselyn id=1)



Vastauksen lähetystä varten

/vastaukset

RequestMethod.POST eli tallennusta varten

Tähän osoitteeseen voi lähettää kaikkkia vastaustyyppejä ja ne tallentuvat tietokantaan

Teksti

{"vastausSisus": "Tarvitaan edullisempaa kahvia", "kysymys":{"kysymysId": 5}}

Checkbox

{"kysymys":{"kysymysId": 4}, "valitutVastaukset": [{ "valittuSisus": "Kahvi"},{"valittuSisus": "Tee"}] }

Radio

{"kysymys":{"kysymysId": 3}, "valitutVastaukset": [{"valittuSisus": "Harvoin"}]}



Extrat

/kysymykset/4/vastaukset

numeron kohdalle laitetaan kysymyksen id.

Tästä linkistä näkee kysymyksen ja kaikki kysymykseen annetut vastaukset mikäli niitä on tallennettu.
