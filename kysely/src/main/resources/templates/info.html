Kysely Info

API End Pointit

Kaikkien tietojen hakeminen

/kyselyt/1/kysymykset

RequestMethod.GET eli saa tiedot palvelimelta.

Näkee kyselyn nimen ja kaikki kyselyn kysymykset ja vastaukset mikäli niitä on tallennettu.

Numeron kohdalle laitetaan kyselyn id. (Kahvi-kyselyn id=1)

Vastauksen lähettäminen

/vastaukset

RequestMethod.POST eli tallennusta varten

Tähän osoitteeseen voi lähettää kaikkkia vastaustyyppejä ja ne tallentuvat tietokantaan.

Tästä osoitteesta voi myös hakea kaikki kyselyyn annetut vastaukset.

Teksti

	{
		"kysymys":{
			"kysymysId": 5
			},
			"vastausSisus": "Tarvitaan edullisempaa kahvia"
			}
	}
